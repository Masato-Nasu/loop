# LOOP

3秒だけ撮影して、そのまま**ループする短い映像**を作る PWA です。  
動画を編集するというより、**時間の断片を小さく切り出して残す**ことを目的にしています。

## 実行

https://loop-8jb.pages.dev/

## 特徴

- **前後カメラ切替**
- **カメラ撮影のみ**
- **3秒固定録画**
- 撮影後すぐに**ループ再生**
- フィルター切替
  - Original
  - B&W
  - Soft
  - Contrast
  - Warm
  - Cool
- **WebM保存**
- **JPEG保存**
- **PWA対応**

## 主な機能

- **Camera / Front / Rear**
  - 背面カメラ・インカメを切り替えて起動します

- **Record 3s**
  - 3秒だけ自動で録画します

- **Loop Preview**
  - 撮影後、すぐにループ再生します

- **Filter**
  - 映像の空気感を切り替えます

- **Save WebM**
  - ループ映像を保存します

- **Save JPEG**
  - サムネイル的な静止画を保存します

## 操作方法

1. Camera を押してカメラを開く
2. Record 3s を押す
3. 3秒の録画が終わると、自動でループ再生
4. 必要ならフィルターを切り替える
5. WebM または JPEG で保存する

## 設計方針

- **最初から短い**
- **迷わない**
- **編集より採集**
- **その場の時間を小さく閉じ込める**
- UIはできるだけ少なくし、結果の映像を優先

## 保存形式

### WebM
ループする短い動画として保存します。  
完成した時間の断片をそのまま残したい時に向いています。

### JPEG
ループ映像の1フレームを静止画として保存します。  
他のアプリや Garden に持っていく素材として使えます。

## PWAとして使う

### iPhone / iPad（Safari）
1. ブラウザで開く
2. 共有ボタンを押す
3. **ホーム画面に追加**

### Android（Chrome）
1. ブラウザで開く
2. メニューを開く
3. **アプリをインストール** または **ホーム画面に追加**

## ローカル確認

```bash
python -m http.server 8000
```

その後、ブラウザで以下を開きます。

```text
http://localhost:8000
```

## デプロイ（Cloudflare Pages）

```bash
npx wrangler pages project create loop --production-branch main
npx wrangler pages deploy . --project-name loop
```

## メモ

LOOP は、長い動画を編集するための道具ではなく、  
**短い時間のかけらを採集し、静かに反復させるための道具**として設計しています。
